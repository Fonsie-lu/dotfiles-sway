# What is it

[Nord|https://www.nordtheme.com/] themed Sway Environment for Arch Linux

# How is it done

no prerequisites
Just run

```sh
./Scripts/install.sh
```

# Where does it go

### Includes:

- yay
- wayland, xwayland, pipewire
- sway, waybar, wofi, alacritty
- Nord Themes
- NeoVim + Plugins
