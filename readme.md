# What is it

[Nord](https://www.nordtheme.com/) themed [Sway](https://swaywm.org/) Environment for [Arch](https://archlinux.org/) Linux

# How is it done

no prerequisites
Just run

```sh
./Scripts/install.sh
```

# Where does it go

Includes:

- yay
- wayland, xwayland, pipewire
- sway, waybar, wofi, alacritty
- Nord Themes
- NeoVim + Plugins

Not included:

- Graphic driver
- fancy kernel
